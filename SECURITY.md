# 安全策略 / Security Policy

## 报告漏洞 / Reporting a Vulnerability

**请勿通过公开 issue 报告安全漏洞**——公开 issue 会在修复前就把漏洞暴露给所有人。
**Please do not report security vulnerabilities through public issues.**

请通过 GitHub Security Advisories 私下报告 / Please report privately via GitHub Security Advisories:
<https://github.com/Ike-li/claude-chat-mobile/security/advisories/new>

这是个人项目，按现状（as-is）提供；我会尽力响应，但不承诺修复时限。
This is a personal project provided as-is; best-effort response, no guaranteed timeline.

本项目的安全模型见 / For the project's security model, see [README · 安全模型](README.md#安全模型)。
