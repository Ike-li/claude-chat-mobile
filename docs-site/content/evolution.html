主线阶段

桥接层

Web ↔ Agent SDK ↔ CLI

以 streaming input 与事件映射为中心；server 作为契约层而非业务 agent。

同步

信封 · 缓冲 · catch-up

seq/epoch 信封、2000 条环形缓冲、sync:since 重连补发；断线可恢复。

驾驶权

单驾驶员 + 只读镜像

Web 与终端不可同时写同一会话；CLI 驾驶时 Web 只读追平 transcript，发送路径接管。

安全纵深

token · TOFU · Access

无 token 绑 127.0.0.1；设备指纹审批；可选 Cloudflare Access JWT fail-closed。

工程门禁

check 流水线

ESLint、import 边界、双向事件契约、inventory、文档一致性；AI 会话下防止结构回缠。
