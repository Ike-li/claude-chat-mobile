封面与导读

Claude Chat Mobile 项目全景手册：认识项目、阅读路线与事实基线。

Claude Chat Mobile 是一层默认上锁的转发层：把你本机已登录的 claude CLI，接到手机浏览器。同一会话、同一份 transcript、同样的权限与配置。

定位

终端等价入口

不是远程桌面，也不是 Claude 的重新实现。手机上打字，应与坐在终端前等价。

形态

自托管桥

Node ≥20 · Express · Socket.IO · Agent SDK 驱动本机 CLI。每实例单用户。

手册

按认知顺序

认识 → 演进 → 方法论 → 实现 → 数据 → 运维 → 规范。以 dev 代码为事实基线。

这本手册从哪来

源码分支为 dev（探针时 HEAD 2de707b）。docs/ 在 2026-07-19 后只保留 deployment.md 与生成的 repository-map.md，设计长文已撤。本站按代码与 README / CLAUDE.md 重叙述，不照搬旧文档目录。

事实基线（非自动同步）手册关键数字与契约以探针时的 dev 代码为准（HEAD 2de707b）。dev 继续演进后，本站不会自动重生成；若行为与手册冲突，以当前代码与 scripts/agent-event-contract.js 为准，并应回写本站或记入漂移清单。
范围本手册描述产品源码（dev），不是 gh-pages 宣传站本身。产物落在 gh-pages 的 docs-site/，便于与展示站同分支维护。
