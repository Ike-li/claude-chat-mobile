环境变量参考

从 .env.example 与 parseServerConfig 核对的配置表。

阅读约 10 分钟 第 27 / 28 篇 源自 .env.example · src/server/config.js

配置以 .env.example 与 parseServerConfig / 各模块读取为准。空串在加载期等同未设置（ANTHROPIC_* 另有规则）。

变量默认含义易错
AUTH_TOKEN空 → 仅 127.0.0.1共享密钥LAN/公网必非空
PORT3000监听端口生产勿撞常驻
CCM_DATA_DIR./data控制面根生产外置绝对路径
WORK_DIR$HOME默认 cwd公网应钉具体项目
WORK_DIRS_FILE空JSON 白名单；热加载契约名不是文件名本身
WORK_DIRS空逗号白名单改后须重启
CLAUDE_BINwhich claudeCLI 路径LaunchAgent 常需绝对路径
IDLE_TIMEOUT_MS600000静默挂死中断过短误杀长工具
INSTANCE_IDLE_RECLAIM_MS1800000空闲回收；0=关与 IDLE 正交
APPROVAL_TTL_MS1800000审批 TTLexample 可能未列
NOTIFY_THROTTLE_MS60000通知节流
SESSION_DELETE_QUIET_MS300000永久删安静期
DEV_MODE空面板重启等生产勿开
LOG_STDERR / LOG_INTERACTIONS / DEBUG_SDK_MESSAGES空诊断日志生产勿长开
CF_ACCESS_HOSTNAME / TEAM / AUD空Access 三项全齐才启用缺一则整层关
VAPID_*空Web Push 三项私钥勿入库
NTFY_URL / TOPIC / TOKEN空ntfy勿用公共裸 topic
PUBLIC_URL空→CF host通知深链
WEB_STATUSLINE默认开off 关闭
CLI_STATUSLINE_BRIDGE空off 回滚 SDK-only不自动卸载 wrapper
CLI_STATUSLINE_DIR约定 snapshots 路径CLI 与 server 同值
ANTHROPIC_*仅 shell网关/模型.env 无效
危险LaunchAgent 若 export AUTH_TOKEN= 空串，会挡住 dotenv 填入。空串应视为未设。

环形缓冲 BUFFER_CAP = 2000 为代码常量，无对应 env。
