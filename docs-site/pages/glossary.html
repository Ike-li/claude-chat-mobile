术语表

instance、envelope、mirror、TOFU 等高频词。

阅读约 6 分钟 第 11 / 28 篇

高频专有名词的准确定义。阅读实现章节前建议扫一遍。

术语定义
instanceId进程内 AgentSession 实例 ID（如 inst_N）。换实例会变。
sessionIdClaude CLI 会话 ID，与 transcript 文件名对应。
epoch实例世代号。新实例更大；客户端见变化应重置 seq 去重基线。
seq实例内事件序号，单调递增；用于 sync:since 补发。
envelopeagent:event 外层：{seq,epoch,sessionId,instanceId,cwd,ts,type,payload}。
AgentSessionsrc/agent/agent.js 中长驻 SDK query 的运行时对象。
viewingInstanceId当前「正在看」的实例；影响高频 delta 是否广播。
mirrorReadonlyCLI 驾驶时的 Web 只读锁，锁定输入。
externalDirty捕获到终端对 transcript 的外部写入后的脏标记。
FRESH无 resumeId 的新开会话；不继承 cwd 末实例档位。
resume以已有 sessionId 冷读/续接 transcript。
TOFUTrust On First Use：设备首次须本机批准。
CCM_DATA_DIR控制面数据根（sessions/devices/audit/push 等）。
WORK_DIR默认工作目录（claude cwd）。
settingSourcesSDK 加载 settings 的来源：user / project / local。
canUseToolSDK 权限回调；未放行则弹 permission_request。
permission_request需用户批准的工具调用事件。
sync:since客户端按 lastSeq 请求缓冲回放。
catchUp轮询 jsonl 增量，镜像 CLI 写入。
ring buffer每实例事件环形缓冲，cap 2000。
worktree sessions发现 git worktree 内会话，使 Web 触达「cd 进 worktree 即 resume」的等价物。
statusline bridge可选包装：CLI 驾驶时把 CLI statusline 快照同步给 Web。
effort思考强度；spawn 时注入 SDK options。Web 切档经 user:setEffort → dispose+resume，不可 in-place 改活进程。
auto（权限档）六合法档之一：SDK 模型分类器自动批/拒；另五档 default / plan / acceptEdits / bypassPermissions / dontAsk。
CLI doctor vs UI 体检CLI：scripts/doctor.js 固定 D1–D11。UI：doctor-runtime.js 另含 Access/设备门/VAPID/白名单等，勿混为一谈。
pendingTurns在途轮计数；仅 send 与 result/中断/撤回结算。禁止双扣；封顶配合 cliQueued。
cliQueued已交 CLI 内部队列的一条消息登记；撤回可能走 cancelAsyncMessage。
bgTasks活后台任务表；多子代理并行时刷新活动、防 idle 误杀；stopTask 只停单个后台任务。
视图守卫写路径 ack / catchUp 提交前校验 viewing 快照未变；防切 tab 后旧响应污染新会话。
pendingSendBusy前端发送忙碌锁；须按会话隔离，禁止跨会话污染。
_raceControlRequestcontrol_request 超时赛跑；覆盖 interrupt/setModel/setPermissionMode/stopTask/cancelAsyncMessage。
