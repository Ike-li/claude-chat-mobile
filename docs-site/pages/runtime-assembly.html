启动与组装根

server.js 薄入口与 app.js 组装根的启动时序。

阅读约 12 分钟 第 12 / 28 篇 源自 server.js · src/server/app.js · src/server/config.js

启动路径刻意拆成薄入口与组装根：环境必须在状态模块求值前就绪；listen 前必须清孤儿审批。

server.js（薄入口）

loadRuntimeEnvironment()   // dotenv；ANTHROPIC_* 只认 shell
installLogTimestamps()     // 须在动态 import 之前
await import('./src/server/app.js')
export httpServer, io, port

约 9 行。LaunchAgent / systemd / npm start / 集成测试都走这里。

环境加载要点

  • dotenv 在动态 import app 之前执行，保证 CCM_DATA_DIR 等在模块求值期已正确（状态模块在 import 时解析路径）。
  • 空串环境变量在加载前删除（除 ANTHROPIC_*），避免 LaunchAgent 的 AUTH_TOKEN= 挡住 .env。
  • ANTHROPIC_* 只认启动 shell 中已有的真实值；写入 .env 的会被剥除。
  • parseServerConfig：port / authToken / idleTimeoutMs / instanceIdleReclaimMs / approvalTtlMs / notifyThrottleMs / sessionDeleteQuietMs / devMode / workDir / dataDir。

app.js 组装顺序

  1. 顶层 import 完成（agent / auth / sessions / files / ops / server 子模块）。
  2. initCfAccess()；parseServerConfig 得到 port/token/timeouts/WORK_DIR/DATA_DIR。
  3. 通知通道 createNotifyChannels（Web Push + ntfy，尽力而为）。
  4. preflight（致命则 process.exit(1)）：
    • WORK_DIR 存在且为目录 → realpathSync
    • 构建 workDirs 白名单（WORK_DIRS_FILE 优先，否则 WORK_DIRS）
    • 解析 CLAUDE_BIN 或 which claude，文件须存在
    • 采集 versions（非致命）；无 ANTHROPIC_AUTH_TOKEN/API_KEY 仅 warn
  5. 初始化 viewing 态：viewingInstanceId=null，viewingCwd=WORK_DIR（不自动 resume）。
  6. Express：configureHttpShell（压缩、CSP、asset ?v= 哈希、静态 public/）。
  7. HTTP 鉴权 createHttpAuth + 运维路由 /health /metrics /push/*。
  8. createServer + Socket.IO；createDeviceGate；io.use 握手鉴权。
  9. createInstanceManager + 连接 handler（业务 on + 文件 handler）。
  10. 可选 WORK_DIRS_FILE 时 fs.watch 热加载。
  11. 进程兜底 uncaughtException / unhandledRejection（只记不退）。
  12. listen 前：expireOrphanedPending + startApprovalRetentionSweep。
  13. listen(port, AUTH_TOKEN ? '0.0.0.0' : '127.0.0.1')；EADDRINUSE → 可读错误 + exit(1)。
  14. SIGINT/SIGTERM：flush sessions/approvals/audit → dispose 全部 agent → io.close → exit。
附注app.js 是有意保留的组装根本体：mirror / catchUp / openInstance 与契约路由共享顶层可变状态（viewing*、mirror*、catchUp*）。域逻辑已下沉，接线不拆成多文件穿针。边界守卫：server 是 sink。
留意无 AUTH_TOKEN 时只绑定 127.0.0.1，手机与隧道均不可达。这是有意设计，不是故障。
