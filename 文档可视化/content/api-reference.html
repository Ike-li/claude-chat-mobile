方法路径鉴权说明
GET/healthhttpAuth健康检查，返回 {status, sessionId, busy, versions, timestamp}
GET/push/vapid-public-keyhttpAuth取 Web Push VAPID 公钥（前端订阅用）

Socket.IO 入向事件（client → server）

会话内操作 user:*

事件payload说明
user:message{text, attachments?, model?, instanceId?, cwd?}发消息。校验非空、text ≤ 50000
user:approve{requestId, decision, alwaysThisSession?}审批挂起的 permission_request
user:answer{requestId, optionIndex}回答 AskUserQuestion
user:interrupt{instanceId?}中断当前轮（不毁会话）
user:setPermissionMode{mode}切权限档：default/plan/acceptEdits/bypassPermissions/dontAsk
user:setEffort{level}切思考强度；busy 时拒切
user:setWorkdir{cwd}切工作区（白名单校验，防穿越）
user:setViewing{instanceId}切当前查看 tab

会话与 tab 管理 session:*（均带 ack）

事件payloadack 返回
session:new{cwd?}{ok, instanceId:null, sessionId:null}
session:switch{sessionId, cwd?}{ok, instanceId, sessionId} 或 {ok:false, error}
session:close{instanceId}{ok, viewingInstanceId} 或 {ok:false, error}
session:list{cwd?, all?}{currentSessionId, sessions[], hasMore}
session:history{sessionId, cwd?}{messages[]} 或 {messages:[], error}

同步/日志/元数据（均带 ack）

事件说明
sync:since断线重连补发环形缓冲；found:false → 客户端清屏重载历史
logs:get该实例的交互日志（需 LOG_INTERACTIONS=1）
models:get当前查看 cwd 的可用模型清单
dev:restart仅 DEV_MODE=1：优雅退出，靠 KeepAlive 自动拉起
