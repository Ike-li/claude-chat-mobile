Claude Chat Mobile

把本机 Claude Code 接到手机上——终端等价性的完整实现

v1.2.1 AGPL-3.0 Node.js + Socket.IO + PWA

这本手册回答一个问题：这个项目怎么工作、为什么这样设计、以及你如何接手。

这是什么

Claude Chat Mobile 是一个自托管的移动端聊天 Web UI，把本机的 claude CLI（Claude Code）接到手机上。核心目标只有一句话："坐在电脑前对 claude 打字"和"在手机上打字"，效果一样。

它不是新的 AI，不是 Claude 的重新实现，也不是远程桌面。它是你本机 Claude Code 的一个移动端入口——同一个 agent、同一份 CLAUDE.md、同样的 MCP 服务器、技能、hooks 和已登录会话。
