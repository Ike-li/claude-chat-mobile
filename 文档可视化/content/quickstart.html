前提

Node.js ≥ 20、claude CLI 已安装并登录（claude --version 输出正常）。

三步启动

  1. 克隆仓库：git clone https://github.com/Ike-li/claude-chat-mobile.git && cd claude-chat-mobile
  2. 配置环境：运行 node scripts/setup.js 交互式配置向导。最小 .env 只需两行：AUTH_TOKEN=your-token 和 WORK_DIR=/path/to/projects。
  3. 启动服务：npm start。浏览器访问 http://localhost:3000。

访问方式

方式地址适用场景
本机http://localhost:3000开发调试
局域网http://<lan-ip>:3000/#token=<TOKEN>家里/办公室同一 Wi-Fi
临时公网cloudflared tunnel --url http://localhost:3000快速测试

trycloudflare.com 临时域名适合短期测试。生产部署请走正式 Cloudflare Tunnel + Access 鉴权。

启动前自检

node scripts/doctor.js 执行 10 项检查（AUTH_TOKEN、CLAUDE_BIN、WORK_DIR、PORT、配置权限等）。全部通过后方可正常启动。
